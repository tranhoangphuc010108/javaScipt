Giải thích độ ưu tiên CSS:

  1. Inline style (style="color: #333;") - Độ ưu tiên cao nhất
  2. ID selector (#main-title) - Độ ưu tiên rất cao
  3. Class selector (.title) - Độ ưu tiên cao
  4. Element selector (h1) - Độ ưu tiên thấp
  5. !important - Ghi đè tất cả (không khuyến khích)

Quy tắc tính điểm độ ưu tiên:

  • Inline style: 1000 điểm
  • ID selector: 100 điểm
  • Class selector: 10 điểm
  • Element selector: 1 điểm
  • Universal selector (*): 0 điểm
