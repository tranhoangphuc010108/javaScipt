#BEM là tiêu chuẩn đặt tên class trong css # ý nghĩa: - bem viết tắt của Block Element Modifier nghĩa tiếng việt là khối phần tử thay đổi - block là khối - element là phần tử - modifier là bổ sung ý nghĩa cho block hoặc element #vì sao phải dùng bem: - giúp viết css dễ dàng hơn - giúp viết css dễ sửa chữa hơn - giúp viết css dễ bảo trì hơn - giúp viết css dễ tái sử dụng hơn - giúp viết css dễ tối ưu hơn - giúp viết css dễ tối ưu hơn - tránh trường hợp khi team có nhiều người cùng viết css vào cùng 1 file css sẽ dễ bị lỗi ## cú pháp: .block__element--modifier có nghĩa là: khối block có phần tử element và có thể thay đổi modifier
