Background Origin Demo

Padding-Box (Mặc định)
Border-Box (Tràn viền)
Content-Box

Giải thích Background-Origin:

  • padding-box (mặc định): Hình nền bắt đầu từ padding, không hiển thị dưới border
  • border-box: Hình nền bắt đầu từ border, tràn ra ngoài viền
  • content-box: Hình nền chỉ hiển thị trong content area

Code để hình nền tràn viền:

.box {
    width: 500px;
    height: 500px;
    background-image: url(https://fullstack.edu.vn/assets/images/f8_avatar.png);
    background-repeat: no-repeat;
    background-size: 100%;
    border: 20px dashed purple;
    padding: 20px;
    box-sizing: border-box;
    background-origin: border-box; /* Để hình nền tràn viền */
}
