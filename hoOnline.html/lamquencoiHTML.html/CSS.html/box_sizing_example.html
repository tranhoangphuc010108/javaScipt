Box Sizing: Border-Box Demo

Content-Box (Mặc định)
Width: 200px + 40px padding + 10px border = 250px thực tế
• Width chỉ tính phần content
• Padding và border được cộng thêm
• Kích thước thực: 250px
Border-Box
Width: 200px (bao gồm padding và border)
• Width bao gồm padding và border
• Kích thước thực: 200px
• Dễ tính toán hơn

Tác dụng của Border-Box:

  • Dễ tính toán: Width và height đã bao gồm padding và border
  • Responsive design: Không bị overflow khi thêm padding/border
  • Layout ổn định: Kích thước không thay đổi khi thêm style
  • Grid/Flexbox: Hoạt động tốt hơn với layout systems

Cách sử dụng:

/* Áp dụng cho tất cả elements */
* {
    box-sizing: border-box;
}

/* Hoặc áp dụng cho từng element */
.my-element {
    box-sizing: border-box;
    width: 200px; /* Bao gồm padding và border */
    padding: 20px;
    border: 5px solid black;
}
