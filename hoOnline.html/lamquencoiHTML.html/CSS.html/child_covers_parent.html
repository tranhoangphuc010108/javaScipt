Thẻ Con Phủ Toàn Bộ Thẻ Cha

Thẻ cha (500x250px):

Code để thẻ con phủ toàn bộ thẻ cha:

HTML:
<div class="box">
  <div class="box-child"></div>
</div>
CSS:
.box {
  width: 500px;
  height: 250px;
  position: relative;
  background-color: #ccc;
}

.box-child {
  width: 80px;
  height: 80px;
  background-color: rgba(128, 0, 128, 0.5);
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
}

Giải thích:

  • position: relative trên thẻ cha: Tạo điểm tham chiếu cho thẻ con
  • position: absolute trên thẻ con: Định vị tuyệt đối
  • top: 0, left: 0, right: 0, bottom: 0: Kéo dãn thẻ con phủ toàn bộ thẻ cha
  • Thẻ con sẽ có kích thước bằng thẻ cha (500x250px)
