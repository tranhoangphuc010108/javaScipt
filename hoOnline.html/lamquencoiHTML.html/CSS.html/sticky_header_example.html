Thuộc tính `position: sticky` và `top: 0`

Cuộn trang xuống để xem hiệu ứng. Header sẽ "dính" lại ở trên cùng của màn hình.

CSS cho Header:
.header {
  width: 100%;
  height: 40px;
  background-color: #000;
  position: sticky;
  top: 0;
}

Giải thích:

  • position: sticky;: Giữ cho element hoạt động như `position: relative` cho đến khi nó cuộn đến một vị trí nhất định, sau đó nó sẽ hoạt động như `position: fixed`.
  • top: 0;: Xác định vị trí mà element sẽ "dính" lại. Trong trường hợp này là ở trên cùng của viewport.
