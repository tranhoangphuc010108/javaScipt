Tác dụng chính của Position Absolute:

  • Thoát khỏi luồng document: Element không chiếm không gian trong layout
  • Định vị chính xác: Có thể đặt element ở bất kỳ vị trí nào
  • Overlay: Tạo lớp phủ lên element khác
  • Modal/Popup: Tạo cửa sổ popup
  • Tooltip: Tạo tooltip hiển thị khi hover
  • Fixed positioning: Element cố định ở một vị trí

Cách sử dụng:

/* Container cha phải có position: relative */ .container { position: relative; width: 400px; height: 200px; } /* Element con sử dụng position: absolute */ .absolute-element { position: absolute; top: 10px; /* Khoảng cách từ trên */ left: 10px; /* Khoảng cách từ trái */ right: 10px; /* Khoảng cách từ phải */ bottom: 10px; /* Khoảng cách từ dưới */ }

Lưu ý quan trọng:

  • Container cha: Phải có `position: relative` để làm điểm tham chiếu
  • Z-index: Có thể sử dụng để kiểm soát thứ tự hiển thị
  • Responsive: Cần cẩn thận khi sử dụng trên mobile
  • Accessibility: Đảm bảo element vẫn có thể truy cập được
