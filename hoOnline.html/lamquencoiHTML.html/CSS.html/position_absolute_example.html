Thẻ Con Phủ Toàn Bộ Thẻ Cha bằng Position

Kết quả:

Code CSS hoàn chỉnh:

Thẻ cha:
.box {
  width: 500px;
  height: 250px;
  position: relative;
  background-color: #ccc;
}
Thẻ con:
.box-child {
  width: 80px;
  height: 80px;
  background-color: rgba(128, 0, 128, 0.5);
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
}

Giải thích thuộc tính position:

  • position: relative trên thẻ cha: Tạo điểm tham chiếu cho thẻ con
  • position: absolute trên thẻ con: Định vị tuyệt đối so với thẻ cha
  • top: 0, left: 0, right: 0, bottom: 0: Kéo dãn thẻ con phủ toàn bộ thẻ cha

Kết quả:

  • Thẻ con sẽ có kích thước 500x250px (bằng thẻ cha)
  • Màu purple trong suốt phủ lên màu xám của thẻ cha
  • Thẻ con phủ toàn bộ thẻ cha
